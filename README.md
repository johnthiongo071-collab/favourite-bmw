## 📖 About This Project

This project combines a personal dream with data — analysing **BMW global sales data from 2010 to 2024** to identify trends, top-performing models, and insights that feed into a very specific mission: **finding the perfect BMW**.

The dream car criteria (non-negotiable):
- ✅ Saloon body style
- ✅ Manual transmission
- 🤔 Everything else is open to persuasion

---

## 🎯 Objectives

- Analyse BMW sales performance across a 14-year period (2010–2024)
- Identify best-selling models and regional trends
- Explore how transmission types and body styles have evolved over time
- Build a data-driven case for the ultimate dream BMW

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/johnthiongo071-collab/favourite-bmw.git
   cd favourite-bmw
   ```

2. **Install dependencies**
   ```bash
   pip install pandas matplotlib seaborn jupyter
   ```

3. **Launch the notebook**
   ```bash
   jupyter notebook bmw.ipynb
   ```
